주차별 출석 체크 (권한 부여)

칸을 클릭하면 해당 학생의 그 주차 영상·자료가 열립니다(🔓). ⭐ 열은 방학 특강 수강 여부입니다.

학생 추가

학생이 로그인 화면에서 입력할 이름과 정확히 같아야 합니다.

주차별 콘텐츠 (학습내용 · 과제 · 교재)

안내문을 그대로 붙여넣으면 학생 화면에서 유튜브/링크가 자동 변환됩니다.

현재 진행 중인 주차

캐릭터(러너)가 멈춰 서고 강조 표시되는 주차입니다.

자료실 폴더

학생 화면 자료실에 보이는 폴더입니다. 이름을 바꾸면 자료·권한도 함께 따라갑니다.

폴더별 학생 권한

‘전체공개’가 켜지면(🔓) 로그인한 모든 학생이 봅니다. 끄면 체크한 학생만 그 폴더를 열람합니다.

교재(책) 등록 — 서재

PDF를 업로드하면 저장소 books/ 폴더에 자동으로 올라가고 링크가 자동 입력됩니다. (GitHub 토큰 1회 필요) · 재생목록·답안 파일은 [링크 버튼]으로 넣으세요. 업로드 후 하단 [GitHub에 저장]도 눌러 목록을 반영하세요.

🎬 프리미엄 설명회 (영상)

유튜브/Vimeo 단일영상 링크를 넣으면 학생 화면에 썸네일 캐러셀 + 목록으로 누적됩니다. 썸네일은 비우면 유튜브에서 자동 추출돼요. (재생목록 링크는 임베드가 안 되니 단일영상 권장)

📣 신규 강좌 공지

제목·본문·(선택)링크. 본문 안 링크는 자동 버튼이 됩니다.

🩺 결석 · 클리닉 신청

학생 버튼이 연결될 네이버 폼(또는 카톡) 주소와 안내문을 넣으세요. 주소가 비면 학생 화면에 ‘준비중’으로 표시됩니다.

🤖 Gemini API 키

AI 자동 생성에 쓰는 원장님 개인 키입니다. 이 브라우저에만 저장되며 사이트에 올라가지 않습니다. (aistudio.google.com/app/apikey 에서 발급)

✒️ DOCSSAM의 프리미엄 칼럼

원장님 칼럼. 주제만 넣고 AI로 초안을 만들거나, NotebookLM 요약을 붙여넣어 정리하거나, 직접 쓰세요. 본문은 HTML도 가능(생성기 결과 그대로 붙여넣기). 학생 VIP 라운지 칼럼 게시판에 표시됩니다.

📰 교육 매거진 (뉴스)

매일 오전 자동으로 대치동 유·초·중등 교육 뉴스가 여기에 쌓입니다(예약 작업). 직접 추가·수정·삭제도 가능하고, 키워드로 즉시 AI 수집도 됩니다.

📊 학생별 성적표

학생을 선택해 전화번호 뒷 4자리(잠금)와 모의고사 성적을 입력하세요. 학생 화면 → 👑 VIP 라운지 → 내 성적표에서 본인만 열람합니다. (뒷자리가 비면 잠금 없이 열립니다)
