학생 추가

학생이 로그인 화면에서 입력할 이름과 정확히 같아야 합니다.

🌟 특강(활용 모의고사) 회차별 수강 권한

특강은 일부 회차만 신청한 학생이 있을 수 있어, 회차별로 볼 수 있는 학생을 따로 정합니다. 전체공개를 끄면 체크한 학생만 그 회차를 봅니다. 모두 체크(☑)를 누르면 전원이 개별 체크로 풀리므로 빼고 싶은 사람만 해제하면 됩니다. (특강 자체를 볼 수 있는지는 ② 학생 명단의 [선택] 특강 등록 여부가 먼저 결정하고, 여기서는 그중 어떤 회차인지를 정합니다)

주차별 콘텐츠 (학습내용 · 과제 · 교재)

안내문을 그대로 붙여넣으면 학생 화면에서 유튜브/링크가 자동 변환됩니다. · 텍스트를 드래그하면 꾸미기 버튼(굵게·형광펜·색·별)이 떠요. · 교재 PDF는 웹 링크 또는 로컬 PDF 파일 선택/드래그로 자동 업로드됩니다. (업로드엔 GitHub 토큰 필요)

현재 진행 중인 주차

캐릭터(러너)가 멈춰 서고 강조 표시되는 주차입니다.

🔑 GitHub 토큰

저장·PDF 업로드에 쓰는 개인 토큰입니다. 이 브라우저에 계속 저장되며(사이트엔 안 올라감), 저장 실패가 나도 자동으로 지워지지 않습니다. 공용 PC에서는 저장하지 마세요.

자료실 폴더

학생 화면 자료실에 보이는 폴더입니다. 이름을 바꾸면 자료·권한도 함께 따라갑니다.

폴더별 학생 권한

‘전체공개’가 켜지면(🔓) 로그인한 모든 학생이 봅니다. 끄면 체크한 학생만 그 폴더를 열람합니다. 새로 만든 폴더는 비공개로 시작하므로, 공개하려면 🔓를 켜거나 학생을 체크하세요.

교재(책) 등록 — 서재

PDF를 파일 선택 또는 드래그하면 저장소 books/ 폴더에 자동으로 올라가고 링크가 자동 입력됩니다. (GitHub 토큰 1회 필요) · 재생목록·답안 파일은 [링크 버튼]으로 넣으세요. 업로드 후 하단 [GitHub에 저장]도 눌러 목록을 반영하세요.

🔑 VIP 메뉴별 열람 권한

‘전체공개’가 켜지면(초록) 로그인한 모든 학생이 그 메뉴를 봅니다. 끄면 체크한 학생만 봅니다. 모두 체크(☑)를 누르면 전원이 개별 체크로 풀리므로, 빼고 싶은 사람만 해제하면 됩니다. 온라인 회원은 이름 아래 배지로 표시됩니다. (학생 유형은 ② 학생 명단에서 전환)

🎬 프리미엄 설명회 (영상)

유튜브/Vimeo 단일영상 링크를 넣으면 학생 화면에 썸네일 캐러셀 + 목록으로 누적됩니다. 썸네일은 비우면 유튜브에서 자동 추출돼요. (재생목록 링크는 임베드가 안 되니 단일영상 권장)

📣 신규 강좌 공지

제목·본문·(선택)링크. 본문 안 링크는 자동 버튼이 됩니다. · 본문 텍스트를 드래그하면 꾸미기 버튼이 떠요.

🩺 결석 · 클리닉 신청

학생 버튼이 연결될 네이버 폼(또는 카톡) 주소와 안내문을 넣으세요. 주소가 비면 학생 화면에 ‘준비중’으로 표시됩니다.

🤖 Gemini API 키

AI 자동 생성에 쓰는 원장님 개인 키입니다. 이 브라우저에만 저장되며 사이트에 올라가지 않습니다. (aistudio.google.com/app/apikey 에서 발급)

✒️ DOCSSAM의 프리미엄 칼럼

원장님 칼럼. 주제만 넣고 AI로 초안을 만들거나, NotebookLM 요약을 붙여넣어 정리하거나, 직접 쓰세요. 본문은 HTML도 가능(생성기 결과 그대로 붙여넣기). 학생 VIP 라운지 칼럼 게시판에 표시됩니다.

📰 교육 매거진 (뉴스)

매일 오전 자동으로 대치동 유·초·중등 교육 뉴스가 여기에 쌓입니다(예약 작업). 직접 추가·수정·삭제도 가능하고, 키워드로 즉시 AI 수집도 됩니다.

📊 학생별 진단 결과 · 코멘트

학생을 선택해 전화번호 뒷 4자리(잠금)·점수·코멘트를 입력하고, 성적표·복습용 시험지 PDF/사진과 풀이 영상 링크를 첨부하세요(파일 선택 또는 드래그). 진단 이름은 자유롭게 직접 입력합니다. 학생 화면 → 👑 VIP 라운지 → 진단 결과에서 본인만 열람합니다. (뒷자리가 비면 잠금 없이 열립니다) · 업로드엔 GitHub 토큰이 필요합니다.

🤖 AI 일일테스트 진단 — 시험 세트

시험별로 영상·스크립트·프롬프트·문항수·장수를 저장해두면 다음에 같은 시험을 볼 때 그대로 재사용됩니다. 학생당 장수를 넣으면 반 전체를 순서대로 올려도 정확히 학생별로 나눕니다(권장). 비우면 첫 장 이름으로 자동 분리합니다.

📸 반 시험지 업로드 → 자동 채점·코멘트

위 세트를 선택하고 반 학생 시험지를 순서대로 한 번에 올리세요(파일 선택 또는 드래그, 여러 장 가능). 시험지 부분만 자동으로 크롭한 뒤, 각 학생 첫 장의 이름으로 분류하고 동그라미/점수 표기를 세어 득점·코멘트를 학생별로 생성합니다. 결과를 검토·수정한 뒤 저장하세요. (Gemini 키 필요)
무료 넉넉 gemini-2.5-flash · 최고품질(결제) gemini-2.5-pro
